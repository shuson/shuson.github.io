Block (the company behind Square and Cash App) has open-sourced Buzz, a self-hostable workspace where humans and AI agents collaborate in the same virtual room — think Discord meets a Git forge, with every interaction signed as a Nostr event on a relay you own. It's an ambitious reimagining of what a developer workspace looks like in an AI-first world, and it's racked up over 6,800 stars since its launch.

Built entirely in Rust, Buzz runs as a Nostr relay where every message, reaction, workflow step, code review, and git event is cryptographically signed and stored as a single event log. This architecture gives it an identical identity model and audit trail whether the author is a person or an autonomous process — agents aren't second-class citizens, they're full participants with keys, reputations, and verifiable histories. Under the hood it's composed of a growing collection of Rust crates spanning desktop, web, and mobile clients, with a migration system for database evolution and benchmarks for performance at scale. Stars: 6,849 ⭐ (+551 forks).

Feature highlights include agents that can open repositories, send patches, and participate in code review directly within the workspace, alongside a desktop app, web interface, and mobile client. The single-relay setup ships today and can be self-hosted, while operators can serve multiple communities across domains. It comes with admin tooling, performance profiling infrastructure, and an extensible plugin model that looks purpose-built for the emerging ecosystem of AI coding agents like Claude Code, Codex, and Aider.

What makes Buzz trend-worthy isn't just the Block pedigree — it's the bet that the future of software development isn't a chat window next to an IDE, but a sovereign workspace where agents and humans share the same ground truth. In a landscape filling up with AI coding tools, Buzz is staking a claim that the platform layer matters just as much as the model. URL: https://github.com/block/buzz
