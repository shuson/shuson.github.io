SimpleX Chat holds the #1 GitHub Trending spot for a third consecutive day, solidifying its status as the most compelling privacy-first messaging platform in open source. The Haskell-based project delivers the first and only messaging network that operates entirely without user identifiers — no phone numbers, usernames, or email addresses.

Key features and stats. Stars: 14,973 ⭐ (+1,180 today). Forks: 865. The platform's unique approach uses ephemeral pairwise identifiers that rotate with every message exchange, rendering metadata surveillance virtually impossible. Encrypted audio/video calls, file transfers, and group chats are all supported — with no central server ever having access to who communicates with whom. The surge in interest reflects growing developer demand for truly anonymous communication infrastructure.
