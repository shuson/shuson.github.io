What began as a Reddit discussion about AI agent specialization has grown into a 123,000-star phenomenon: The Agency, by creator msitarzewski, is an open-source collection of 232 meticulously crafted AI agent personalities spanning 16 professional divisions — from engineering and game development to digital marketing, finance, cybersecurity, and even GIS mapping. Unlike generic "act as a developer" prompts, each agent is a fully fleshed-out persona with its own voice, battle-tested workflows, concrete deliverables, and measurable success metrics. The project fills a crucial gap in the AI tooling ecosystem: while LLMs are remarkably capable, their output quality varies dramatically depending on how you prompt them. agency-agents solves this by packaging deep domain expertise into portable, reusable Markdown files that can be dropped directly into Claude Code, Cursor, Codex, Gemini CLI, OpenCode, OpenClaw, Windsurf, Aider, Kimi Code, Qwen Code, Osaurus, and Hermes — effectively giving every developer an on-demand team of specialists who never sleep, never complain, and always ship.

The architecture is elegantly simple yet remarkably effective. Each agent lives as a Markdown file in a category directory (engineering/, design/, marketing/, security/, and so on), with structured YAML frontmatter and sections for identity, core mission, critical rules, technical deliverables with code examples, workflow processes, and success metrics. The project is primarily written in Shell (for its installation and conversion scripts) but the real "code" is the prompt engineering craft embedded in each agent file — a testament to the emerging discipline of prompt-as-software. A native desktop app (for macOS, Linux, and Windows) provides one-click browsing and installation, while a parallelized conversion pipeline (./scripts/convert.sh --parallel) transforms the canonical Markdown agents into each target tool's native format: .mdc rules for Cursor, TOML for Codex, SKILL.md for Antigravity and Osaurus, YAML for Kimi Code, and lazier-router plugins for Hermes. With 123,347 ⭐ and 20,073 forks, the project has also spawned community translations in Chinese, Japanese, Portuguese, Russian, Indonesian, Arabic, and Korean. Licensed under MIT.

Standout capabilities run deep and wide. The Engineering Division alone features 30+ agents covering everything from Frontend Developer (React/Vue/Angular with Core Web Vitals optimization) and Embedded Firmware Engineer (bare-metal RTOS on ESP32/STM32/Nordic) to Solidity Smart Contract Engineer, Incident Response Commander, and a Codebase Onboarding Engineer that reads unfamiliar repos and traces code paths to onboard developers fast. The Marketing Division fields 40+ specialists including a Reddit Community Builder, Bilibili Content Strategist, Carousel Growth Engine, China E-Commerce Operator, and an AI Citation Strategist targeting visibility across ChatGPT, Claude, Gemini, and Perplexity. The Game Development Division covers Unity, Unreal Engine, Godot, Roblox Studio, and Blender with engine-specific agents like Unreal Systems Engineer (C++/GAS/Nanite) and Godot Gameplay Scripter (GDScript 2.0 with composition patterns). A nascent GIS Division brings spatial computing to the mix with GeoAI/ML Engineer for feature extraction and 3D Scene Developer for Cesium-based digital twins. Each agent comes with "when to use" guidance and real-world scenario blueprints — from startup MVP assembly and multi-channel marketing campaigns to smart campus digital twin pipelines and enterprise feature development with full quality gates.

The project's explosive rise reflects a broader shift in how developers think about AI tooling. The industry is moving past the "single-prompt" era into an agentic paradigm where teams of specialized AI personas collaborate on complex workflows — and agency-agents provides the most comprehensive, community-vetted library of these personas to date. It resonates because it doesn't try to be another AI platform or wrapper; it simply gives developers battle-tested characters they can plug into whatever tool they already use. With 357 commits, active discussions, and a roadmap that includes an interactive agent selector and community marketplace, The Agency is evolving from a passion project into essential infrastructure for the agentic coding era. Find it at https://github.com/msitarzewski/agency-agents
