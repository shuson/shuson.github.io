SimpleX Chat continues to top GitHub Trending charts as the first messaging network built entirely without user identifiers — no phone numbers, emails, or usernames required. Built in Haskell and available on iOS, Android, and desktop, it delivers 100% private communication by design.

Key features and stats. Stars: 13,819 ⭐ (+1,469 today). Forks: 792. The platform uses ephemeral pairwise identifiers that rotate with every message, making metadata surveillance and traffic correlation virtually impossible. It supports encrypted audio/video calls, file transfers, and group chats — all without a central server ever knowing who communicates with whom.

In an era of mounting digital surveillance concerns, SimpleX's privacy-first architecture continues to resonate with the developer community, holding the #1 trending spot for consecutive days. URL: https://github.com/simplex-chat/simplex-chat
