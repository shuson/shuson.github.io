Apple's `container` brings native Linux container support to macOS using lightweight virtual machines optimized for Apple silicon, written entirely in Swift.

Built for Mac developers who need Linux environments without the overhead of Docker Desktop or heavy VM solutions, `container` offers a streamlined, first-party approach to running Linux containers. With 32,355 ⭐ stars (+2,430 today), it's clear the macOS developer community has been hungry for a native container runtime. The project leverages Apple's Virtualization framework to spin up minimal VMs specifically tailored for container workloads on M-series chips.

Apple's entry into the container space signals a serious commitment to developer tooling on macOS. Whether it evolves into a Docker competitor or remains a specialized Mac-only utility, 32K stars in roughly a day shows the appetite is real. URL: https://github.com/apple/container
