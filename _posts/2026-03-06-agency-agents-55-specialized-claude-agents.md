## 🎭 Agency Agents

**Agency Agents** is a comprehensive collection of 55+ specialized AI agent personalities designed for Claude Code. Born from a Reddit discussion about AI agent specialization, this project delivers what generic "act as a developer" prompts cannot: deep expertise with distinct personality, workflows, and measurable deliverables.

Think of it as assembling your dream team—except they're AI specialists who never sleep, never complain, and always deliver.

### What Makes This Different

Most agent prompts are generic templates. Agency Agents takes a different approach:

| Traditional Prompts | Agency Agents |
|---------------------|---------------|
| "Act as a developer" | Specialized expertise with domain depth |
| Vague guidance | Concrete deliverables with code examples |
| One-off collections | Comprehensive systems with workflows |
| Black box tools | Transparent, forkable, adaptable |
| No success metrics | Measurable outcomes and quality gates |

Each agent includes:
- **Identity & Personality** — Unique voice and communication style
- **Core Mission** — Clear purpose and scope
- **Critical Rules** — Domain-specific constraints and best practices
- **Technical Deliverables** — Real code examples and outputs
- **Workflow Process** — Step-by-step procedures that work
- **Success Metrics** — How to measure quality and completion
