SimpleX Chat is the first messaging network that operates entirely without user identifiers of any kind — no phone numbers, no emails, no usernames. Built with Haskell and available on iOS, Android, and desktop, it delivers 100% private communication by design.

Key features and stats. Stars: 12,533 ⭐ (+432 today). Forks: 709. The platform eliminates traditional identity-based connections, using ephemeral pairwise identifiers that change with every message, making metadata surveillance and traffic correlation virtually impossible. It supports encrypted audio/video calls, file transfers, and group chats — all without a central server knowing who is talking to whom.

In an era of increasing digital surveillance, SimpleX represents a radical privacy-first approach that rethinks messaging from the transport layer up. URL: https://github.com/simplex-chat/simplex-chat
