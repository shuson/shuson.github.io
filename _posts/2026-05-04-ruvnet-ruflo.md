Ruflo (by ruvnet) is the leading agent orchestration platform for Claude Code — deploy intelligent multi-agent swarms, coordinate autonomous workflows, and build conversational AI systems.

With over 40k stars and 32+ plugins, it adds coordinated swarms, self-learning memory, federated comms, and enterprise security. Features include native Claude Code integration, RAG-powered knowledge retrieval, GPU-accelerated vector search, browser automation, security auditing, and more.

One `init` gives Claude Code a nervous system — agents self-organize, learn from every task, and collaborate across machines. MIT licensed, built with TypeScript and Rust-powered WASM kernels.
